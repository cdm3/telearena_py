# Tele-Arena 5.6 Python Port

A local single-player Python port of **Tele-Arena 5.6**, a classic multi-player RPG originally built for The Major BBS / Worldgroup BBS systems. This port replaces the BBS multi-user infrastructure with a curses terminal UI and local save files, while preserving the original gameplay, room data, and message text.

## Requirements

- Python 3.8+
- A terminal with color support (macOS Terminal, iTerm2, Linux xterm, etc.)
- The original Tele-Arena 5.6 data files in `Ta56dSrc/` (not redistributed)

## Quick Start

```bash
# Parse the original data files into JSON (only needed once)
python3 parse_data.py

# Start the game (default user: player1)
python3 main.py

# Use a different save slot or starting sex
python3 main.py --user mychar --sex f
```

## Controls

| Key | Action |
|-----|--------|
| Type a command + Enter | Submit command |
| Up/Down arrows | Command history |
| Backspace | Delete character |

## Commands

| Command | Aliases | Description |
|---------|---------|-------------|
| `look` | `l` | Describe current room |
| `n`, `ne`, `e`, `se`, `s`, `sw`, `w`, `nw`, `u`, `d` | (direction names) | Move |
| `exits` | `ex` | List exits with destination room numbers |
| `status` | `st` | Show character sheet |
| `health` | `he` | Show HP/SP |
| `inventory` | `i` | Show carried items |
| `get <item>` | `g` | Pick up item |
| `drop <item>` | `dr` | Drop item |
| `wield <item>` | `wi` | Equip weapon |
| `wear <item>` | `we` | Equip armor |
| `attack <target>` | `a` | Attack monster or player |
| `cast <spell>` | `ca` | Cast a spell |
| `buy <item>` | | Purchase from shop |
| `sell <item>` | | Sell to shop |
| `list items` | | List shop inventory |
| `reroll` | | Re-roll stats (level 1 only) |
| `suicide` | | Delete character and start over |
| `help` | `he` | Show command list |
| `exit` | `quit` | Leave the game |

## Project Layout

```
telearena_py/
├── main.py              # Entry point
├── parse_data.py        # One-time data parser (MCV → JSON)
├── game/
│   ├── constants.py     # All game constants (translated from TSGARN-0.H)
│   ├── character.py     # Character struct + generation + save/load
│   ├── engine.py        # Game loop, command dispatch, state machine
│   ├── world.py         # Room/dungeon management
│   ├── messages.py      # MSG file loader + ANSI color parser
│   ├── monsters.py      # Monster types + active instance manager
│   ├── combat.py        # Attack resolution, damage, death
│   ├── shops.py         # Shop buy/sell logic
│   └── ui/
│       └── curses_ui.py # Curses terminal UI
├── data/                # Parsed JSON data (generated by parse_data.py)
├── saves/               # Per-user character save files (JSON)
└── Ta56dSrc/            # Original C source + data files (not redistributed)
```
